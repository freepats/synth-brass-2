Synth Brass #2
--------------

Version 2024-06-10

Samples by Strix SoundFont Team <dsoundfont123@gmail.com>, sent to the FreePats project on June 2018.

This sound set emulates the 'BRASS 7' patch in the Yamaha DX7. Recorded from Dexed[1], a software synthesizer that emulates the sound synthesis algorithm of the DX7.

[1] https://asb2m10.github.io/dexed/
